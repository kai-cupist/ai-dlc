# 인프라 설계 - backend-api

## 인프라 매핑

| 논리적 컴포넌트 | AWS 서비스 | 설정 |
|----------------|-----------|------|
| FastAPI 서버 | **ECS Fargate** | Task 1개, vCPU 0.5, 메모리 1GB |
| 로드 밸런서 | **ALB** | HTTPS 443, 헬스체크 /health |
| PostgreSQL | **RDS PostgreSQL 16** | db.t3.micro, 20GB gp3, 단일 AZ |
| 이미지 저장소 | **S3** | 버킷 1개, 서버 측 암호화 (AES-256) |
| 컨테이너 레지스트리 | **ECR** | 이미지 보관, 라이프사이클 정책 |
| 시크릿 관리 | **Secrets Manager** | DB 비밀번호, JWT 시크릿 키 |
| 로그 수집 | **CloudWatch Logs** | ECS 로그 드라이버, 90일 보관 |
| DNS | **Route 53** (선택) | 커스텀 도메인 연결 |
| TLS 인증서 | **ACM** | ALB에 적용 |

## 보안 그룹

| 보안 그룹 | 인바운드 | 아웃바운드 |
|-----------|---------|-----------|
| **ALB SG** | 0.0.0.0/0:443 (HTTPS) | ECS SG:8000 |
| **ECS SG** | ALB SG:8000 | RDS SG:5432, S3 (VPC Endpoint), 0.0.0.0/0:443 (외부 API) |
| **RDS SG** | ECS SG:5432 | 없음 |

## S3 버킷 정책

| 설정 | 값 |
|------|-----|
| 버킷명 | `{project}-menu-images-{env}` |
| 암호화 | SSE-S3 (AES-256) |
| 퍼블릭 액세스 | 차단 (Block All Public Access) |
| 접근 | ECS Task Role에만 s3:PutObject, s3:GetObject 허용 |
| 이미지 접근 | CloudFront 또는 Pre-signed URL |

## Terraform 모듈 구성

```
backend/infra/
+-- main.tf              # 프로바이더, 백엔드 설정
+-- variables.tf         # 변수 정의
+-- outputs.tf           # 출력 값
+-- modules/
    +-- vpc/             # VPC, 서브넷, NAT, 보안 그룹
    +-- ecs/             # ECS 클러스터, 서비스, 태스크 정의
    +-- rds/             # RDS 인스턴스, 파라미터 그룹
    +-- alb/             # ALB, 타겟 그룹, 리스너
    +-- s3/              # S3 버킷, 정책
    +-- ecr/             # ECR 리포지토리
    +-- secrets/         # Secrets Manager
    +-- logs/            # CloudWatch 로그 그룹
```
